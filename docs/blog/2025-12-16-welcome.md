# Welcome to Synth Studio

We're excited to launch the official documentation for **Synth Studio** - the privacy-preserving synthetic data generation platform.

## What is Synth Studio?

Synth Studio helps you generate high-quality synthetic data with mathematical privacy guarantees. Whether you're working with healthcare data under HIPAA, financial data under GDPR, or any sensitive dataset, Synth Studio ensures your synthetic data is both useful and private.

## Key Features

- **Differential Privacy**: (ε, δ)-DP guarantees with RDP accounting
- **Multiple Generators**: CTGAN, TVAE, GaussianCopula, TimeGAN
- **Quality Evaluation**: Statistical similarity, ML utility, privacy leakage detection
- **Compliance Ready**: HIPAA, GDPR, CCPA, SOC-2 reporting
